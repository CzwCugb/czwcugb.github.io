圣诞彩灯计划

圣诞前夕，小明和妹妹在装饰一棵巨大的电子圣诞树。这棵树的每个枝杈连接处都是一个可编程的节点，能发出红色、蓝色的光芒，或保持熄灭。妹妹提议用游戏决定装饰方案：若小明能计算出在满足特定平衡条件时红色灯的最大数量，就按他的想法布置彩灯，否则妹妹要主导整个设计。

圣诞树的结构被建模为一棵以节点 1 为根的树。每个节点可设置为红灯、蓝灯或保持熄灭。装饰规则要求：对于每个非叶子节点，以它为根的子树中红灯数必须等于蓝灯数。小明需要找到满足该规则时，整棵树最多能点亮多少个红灯。

输入格式

第一行输入正整数 n (1≤n≤1e5) 表示节点总数。 接下来 n−1 行，每行两个整数 u,v (1≤u,v≤n) 描述节点间的连接关系。

输出格式

输出一行整数，表示红灯的最大数量。

数学作业

今天小 X 的数学老师带领大家学习了斐波那契序列：

斐波那契序列指的是这样一个数列：{1,2,3,5,8,13,21,34}。从第 3 个数开始，每个数都是前两个数的和，比如 8=3+5,34=13+21。数列里的数叫做斐波那契数。

一个数 n 的斐波那契表示是指把 n 写成若干个互不相同的斐波那契数的和。一个数可以有多种不同的斐波那契表示。比如 14 有三种斐波那契表示：14=1+13,14=1+5+8,14=1+2+3+8。数学老师给小 X 留下了一个数学作业，她告诉小 X 一个正整数 n，想让小 X 算出 n 有多少种不同的斐波那契表示。

小 X 请你帮助他完成他的数学作业。
