软件工程是系统地应用工程原理来设计、开发和测试软件。

它专注于高效地构建高质量的软件，同时考虑成本和未来的更新。

What’s the biggest challenge / problem in this project ?

The most difficult part of this work is how to represent a program as a feature graph in a suitable way, so that it contains enough semantic information.

But we finally find a solution to construct an specific instance of Code Propety Graph, which combines control flows and syntax structure in a joint structure.

这项工作中最困难的部分是如何以合适的方式将程序表示为特征图，使其包含足够的语义信息。

但是我们最终找到了一个解决方案来构造一个特定的程序属性图实例，它将控制流和语法结构结合在一个联合结构中。

What are the shortcomings of this project research ?

The shortcoming of our approach is that it doesn't work for recursive programs.

There are two kinds of non-termination: one caused by loop, and the other is caused by recursion.

Since our model is built around loop structures, it can't handle recursive instance.

我们方法的缺点在于：它无法适用于递归程序。

有两种非终止，一种来自于循环，另一种来自于递归。

我们基于循环结构设计我们的模型，因此它无法适用于递归程序。

我认为人工智能的快速发展给我们的日常生活带来了极大的便利。
